- 使用`git bash`删除每个目录下的`.gitkeep`文件

```bash
find . -type d -not -path "*/.git*" -exec rm -f {}/.gitkeep \;

```

- 为空文件夹增加`.gitkeep`文件

```bash
find ./ -type d -empty -not -path "./.git/*" -exec touch {}/.gitkeep \;
```
